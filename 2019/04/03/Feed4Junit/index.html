Feed4Junit

Apr 3, 2019

简介

  • 是数据与代码分离的一个单测框架
  • 在Junit测试框架上进行了扩展
  • 支持csv和Excel文件数据源头，甚至支持直接连接数据库

idea引用

  • maven工程增加依赖
  • 引包 import org.databene.feed4junit.Feeder;

参考文档

  • 使用 Feed4JUnit 进行数据与代码分离的 Java 单元测试

采坑

  1. 数据文件一定要放到工程根目录下，不是src目录。否则引不到。
  2. 运行单个case带有@Test标签的报错：

没有找到合适的解决办法，网上查了下直接运行整个Test类就不会报错，试了下确实是这样。
