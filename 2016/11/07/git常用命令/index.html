git常用命令

Nov 7, 2016

clone 克隆项目

  • git clone <git地址> <本地地址>

git push 和 pull 提交代码

  • 进入工程目录下

提交代码老三步：

  • git add . (添加所有文件)
  • git commit -m “本次提交的注释”
  • git push

拉代码就一步：

  • git pull

git 新建、切换、删除分支

  • git branch -a 查看本地分支
  • git checkout -b <分支名> 新建并切换到本地分支
  • git checkout <分支名> 切换到某分支
  • git branch -d <分支名> 删除分支
  • git merge <分支名> 合并某分支到当前分支
